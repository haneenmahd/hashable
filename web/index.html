Hashable

Generate hashes without even adding a script tag

Hashable is an open-source, client-side gateway for users to test out cryptographic functions and hashing methods. We also support a REST API which users can use to hash strings in their own website on client-side Javascript Applications.

Getting Started

It is easy and simple to use our REST API.

There are only three steps in-order to use our server


First of all, you will need to send a get request into our server. You can use fetch API of the browser or use axios or
other third-party tools. Our server is hosted on heroku at https://hashable-server.herokuapp.com.
